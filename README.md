# Air-quality-Prediction-using-ML

Repository reviewing recent advancements in air quality prediction using ML techniques. Covers various models (linear regression, decision trees, neural networks) and data sources (meteorological, pollutant concentrations). Discusses challenges (data scarcity, model interpretability) and future directions integrating ML with sensor tech for enhanced accuracy.
